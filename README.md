# Portafolio jlruedam_dev (React + Vite)

Migración del portafolio estático a una aplicación React con Vite y JavaScript.

## Requisitos

- Node.js 18+
- npm 9+

## Ejecutar en local

```bash
npm install
npm run dev
```

La app abrirá en una URL similar a `http://localhost:5173`.

## Build de producción

```bash
npm run build
npm run preview
```

## Estructura principal

- `src/App.jsx`: layout principal y navegación por pestańas.
- `src/styles.css`: estilos del sitio.
- `public/img`: imágenes migradas desde el proyecto original.
